쿼리스트링 표현방법

- 쿼리스트링 : URL의 뒤에 입력 데이터를 함께 제공하는 가장 단순한 데이터 전달 방법.
웹개발에서 데이터를 요청하는 방식 중 주로 GET방식으로 데이터를 요청할 때 쓰이는 방법.
ex:) https://www.ksmart.or.kr?key=value&key=value - 문법 : @{주소(key=${value},key=${value})} - 예시 : @{/user/detail(userId=${userId},userName=${userName})}
