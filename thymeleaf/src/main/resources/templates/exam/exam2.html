객체와 반복문

- 문법 : th:each="약어 : ${데이터유형: 순회가능한 객체}"
- ex) th:each="l : ${userList}
- 반복범위: 태그 속성(th:each)을 추가했다면 추가한 태그가 닫힐 때까지
- 대상 : 배열, 컬렉션 등 순회가능한 객체 | java 읽기전용 for문 ex) for(User user : userList(대상)){}

아이디 비밀번호 이름 권한 이름 연락처
[[${l.userMobile}]]
객체(배열, 컬렉션)의 인덱스 객체(배열, 컬렉션)의 번호 객체(배열, 컬렉션)의 크기 객체(배열, 컬렉션)의 홀수 여부 반환(boolean) 객체(배열, 컬렉션)의 짝수 여부 반환(boolean) 객체(배열, 컬렉션)의 첫번째 요소 혹은 마지막 요소 여부 반환(boolean)
